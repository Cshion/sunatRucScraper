# Sunat Ruc Scraper

## Requisitos
- Installed Nodejs >= 0.10

## Instalation
- npm install 


## Example :

	var rucScraper = require("./lib");
	rucScraper({ruc:"20434178780"},function  (err,data) {
		if(err)
			console.log(err)
		console.log(data);
	})
